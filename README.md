## Overview
<!-- Write Overview about this project -->
경기 성남 소재 성일고등학교의 급식 메뉴와 실시간 시간표, 학사일정, 가정통신문 등을 빠르게 확인할 수 있는 서비스입니다.
웹 앱(PWA) 형태로 개발되어 모바일 및 데스크톱에서 앱 형태로 설치하여 더욱 빠르게 이용하실 수 있습니다.
